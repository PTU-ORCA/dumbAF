Elective-V Adhoc and Wireless Sensor Networks (BTCS 716-18) - Theory

Course Objectives

This course aims to:

  • Introduce the fundamental concepts and characteristics of ad hoc and wireless sensor networks.
  • Provide an understanding of various routing protocols and algorithms used in these networks.
  • Explore issues related to medium access control, power management, and security in ad hoc and sensor networks.
  • Enable students to analyze and design solutions for challenges in these networks.

Course Outcomes

Upon completion of this course, students will be able to:

  1. Explain the key characteristics and challenges of ad hoc and wireless sensor networks.
  2. Understand and analyze various routing protocols for these networks.
  3. Design and implement medium access control and power management techniques for efficient operation.
  4. Identify and address security vulnerabilities in ad hoc and sensor networks.
  5. Apply their knowledge to design and evaluate solutions for real-world applications.

Detailed Syllabus

Module 1: Introduction

  • Introduction to Ad Hoc Networks, Applications, Characteristics & Challenges
  • Introduction to Wireless Sensor Networks (WSNs), Applications, Characteristics & Challenges
  • Differences between Ad Hoc and Sensor Networks

Module 2: MAC Protocols

  • Issues in Designing a MAC Protocol for Ad Hoc Wireless Networks, Design goals of a MAC Protocol for Ad Hoc Wireless Networks
  • Classification of MAC Protocols, Contention-Based Protocols, Contention-Based Protocols with Reservation Mechanisms, Contention-Based MAC Protocols with Scheduling Mechanisms
  • Hybrid MAC Protocols, Directional MAC Protocols
  • Power Aware MAC protocols.

Module 3: Routing Protocols

  • Issues and Challenges in Designing a Routing Protocol for Ad Hoc Wireless Networks, Classification of Routing Protocols
  • Table-Driven Routing Protocols: Destination-Sequenced Distance-Vector Routing, Wireless Routing Protocol
  • On-Demand Routing Protocols: Ad Hoc On-Demand Distance Vector Routing, Dynamic Source Routing, Temporally Ordered Routing Algorithm, Associativity Based Routing
  • Hybrid Routing Protocols: Zone Routing Protocol
  • Routing Protocols in Wireless Sensor Networks: Sensor Protocol for Information via Negotiation, Directed Diffusion
  • Power Aware Routing Protocols.

Module 4: Transport Layer Protocols

  • Issues and Challenges in Designing a Transport Layer Protocol for Ad Hoc Wireless Networks
  • Classification of Transport Layer Solutions
  • TCP Over Ad Hoc Wireless Networks: Problems of TCP in Ad Hoc Wireless Networks, Solutions for TCP over Ad Hoc Wireless Networks
  • Other Transport Layer Protocols for Ad Hoc Wireless Networks: Ad Hoc Transport Protocol

Module 5: Security

  • Security Attacks in Ad Hoc Wireless Networks & Sensor Networks, Issues and Challenges in Security Provisioning
  • Network Security Requirements, Security in Ad Hoc Wireless Networks, Secure Routing in Ad Hoc Wireless Networks
  • Security in Wireless Sensor Networks
  • Key Management.

Module 6: Quality of Service (QoS) and Energy Management

  • QoS in Ad Hoc Wireless Networks, Issues and Challenges in QoS Provisioning
  • Classification of QoS Solutions, QoS Frameworks for Ad Hoc Wireless Networks
  • QoS in Wireless Sensor Networks, Energy Management in Wireless Sensor Networks

Elective-V Adhoc and Wireless Sensor Networks Lab (BTCS 717-18)

List of Experiments

  1. Implementation of MAC protocols in Adhoc network.
  2. Implementation of any routing protocol in Adhoc network.
  3. Implementation of power aware routing in Adhoc network.
  4. Implementation of DSR in Adhoc network.
  5. Implementation of AODV in Adhoc network.
  6. Implementation of security algorithms in Adhoc Network.
  7. Simulation of various MAC protocols, Routing protocols in wireless sensor network.
  8. Implementation of security algorithms in wireless sensor network.

Course Outcomes

Upon completion of this lab, students will be able to:

  1. Simulate and implement various protocols and algorithms for ad hoc and wireless sensor networks.
  2. Analyze the performance and trade-offs of different solutions.
  3. Gain practical experience in designing and evaluating network protocols.
  4. Use network simulation tools effectively.
